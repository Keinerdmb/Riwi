   <h1 align="center">Cajero automático</h1> 


Este cajero fue creado con la intencion de realizar transacciones financieras de forma rapida y segura sin necesidad de ir a una sucursal. 


---


## Contenido

1. **Clave**: Te muestra tu clave dinamica que cambia cada que vez la solicites.

2. **Consultar movimientos**: Te permite revisar tus Pagos, retiros y transferencias según la fecha en que se realizaron.

3. **Pagos**: Permite realizar pagos de servicios, facturas o recargas directamente desde tu cuenta, indicando el monto y el destinatario.

4. **Retiro**: Permite realizar retiros
